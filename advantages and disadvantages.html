The Advantages and Disadvantages of Operating Systems

  1. The operating system provides the interface between the user and the hardware.
  2. It can be easily used because it has a graphical user interface, and new users can run computers easily by this.
  3. Accessing hardware without writing programs

    The Disadvantages of Operating Systems

  1. Some operating systems are free but some are expensive such as – windows costs around 5000 ₹ to 10000 ₹.
  2. It is a bit difficult to run Linux compared to Windows.
  3. Sometimes the operating system crashes due to which the user’s data is destroyed.
